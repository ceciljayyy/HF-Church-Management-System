# Product Requirements

This platform is a production-ready, multi-branch Church Management System.

## Core capabilities

- Auth and RBAC
- Dashboard and reporting
- People, members, families, groups, and events
- Attendance and kiosk mode
- Finance, contributions, expenses, pledges
- Communications, notifications, media, settings, and audit logs
- Pastoral care, first timers, prayer requests
- Plugin/integration foundation

## Non-functional requirements

- TypeScript end to end
- Secure auth and protected routes
- Soft delete or archive for key entities
- Validation with Zod
- Audit logging for sensitive operations
- Responsive premium UI
- Pagination, filtering, and search

## Design direction

- Dark premium dashboard
- Black/charcoal surfaces
- Lime and emerald accents
- Rounded cards and subtle borders
- Left sidebar, topbar, and right activity panel
